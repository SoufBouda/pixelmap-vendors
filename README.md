# Pixel Map - Vendor Packages

Vendor tool packages for the Pixel Map installer. Each release contains a zip archive for one vendor tool.

## Vendors

| Tag | Tool | Description |
|-----|------|-------------|
| `activitywatch-v*` | ActivityWatch | Window/application tracking |
| `screenpipe-v*` | Screenpipe | OCR screen capture |
| `windrecorder-v*` | Windrecorder | Video screen recording |

## Usage

These packages are downloaded automatically by the Pixel Map Compact installer, or bundled in the Full installer.
